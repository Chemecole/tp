Cette application permet de rechercher des expériences à réaliser de deux manières différentes.

RECHERCHE

Recherche par produits

Sur cette page, choisissez dans le menu les produits que vous souhaitez utiliser. Comme résultat, vous obtiendrez
1) l'emplacement du produit (Armoire A//) ainsi que la quantité disponible,
2) les TPs utilisant ces produits.
Note: les TPs affichés utilisent tous les produits choisis par l'utilisateur. Donc, plus il y a de produits selectionnés, plus le nombre de TPs se réduit.
